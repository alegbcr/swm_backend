Api Strata Wealth Management

Bienvenido a Strata Wealth Management API. Esta es una API desarrollada con Node.js para gestionar la información de los nuevos clientes.

Endpoints

  • Regresa una lista de todos los clientes.

    GEThttps://vps.stratawealthmanagement.com/api/v1/customer-contact

  • Crea un nuevo cliente.

    POSThttps:/vps.stratawealthmanagement.com/api/v1/customer-contact

  • Regresa los detalles de un cliente específico por ID.

    GET https://vps.stratawealthmanagement.com/api/v1/customer-contact/{id}

  • Actualiza la información de un cliente específico por ID.

    PATCHhttps:/vps.stratawealthmanagement.com/api/v1/customer-contact/{id}

  • Elimina un cliente específico por ID.

    DELETEhttps://vps.stratawealthmanagement.com/api/v1/customer-contact/{id}
